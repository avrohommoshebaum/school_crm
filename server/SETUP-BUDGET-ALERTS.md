# Setup Budget Alerts in Google Cloud

## Why?
Protect yourself from unexpected costs. Get notified before bills get high.

## Steps:

1. **Go to Google Cloud Console**
   - Navigate to: Billing â†’ Budgets & alerts

2. **Create Budget**
   ```
   Budget Name: School App Monthly Budget
   Amount: $100/month (adjust as needed)
   ```

3. **Set Alerts**
   - Alert at 50% ($50)
   - Alert at 90% ($90)
   - Alert at 100% ($100)

4. **Actions** (Optional)
   - Disable billing (nuclear option - stops service)
   - Email notifications

## Recommended Budgets:
- **Normal Usage**: $50/month
- **Peak Usage**: $200/month (for 400-user spikes)
- **Emergency**: $500/month (hard limit)
